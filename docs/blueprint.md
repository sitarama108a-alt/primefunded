# **App Name**: PrimeFunded

## Core Features:

- AI-Powered Compliance Monitor: An intelligent tool that analyzes real-time MT5 trading data to ensure adherence to 1-Step, 2-Step, and Instant Funding rules, flagging risk patterns before they trigger a hard breach.
- Live MetaApi Dashboard: A high-performance trading dashboard displaying real-time balance, equity, and drawdown metrics pulled directly from MetaTrader 5 using MetaApi.
- Secure Auth & KYC Onboarding: Streamlined user authentication using Firebase Auth with dedicated portals for login, signup, and account verification status.
- Funding Challenge Hub: Interactive interface for users to browse, select, and purchase various trading challenge tiers and monitor their progress toward funded status.
- Automated Order Processing: Instant verification system that tracks payment submissions to the Firebase Realtime Database and auto-generates MT5 credentials upon admin approval.
- Real-time Notification Suite: Automated email system powered by Resend API to deliver credentials, milestone alerts, and account breach notices immediately to traders.
- Cloud Ledger Sync: Persistent synchronization of order history, user account performance, and trading metrics stored across Firebase nodes.

## Style Guidelines:

- Primary Color: Electrifying Azure (#2D5BFF) used for key action buttons and status indicators to contrast against the dark navy base.
- Background Color: Deep Space Navy (#0A1128) providing a sophisticated, professional institutional trading environment.
- Accent Color: Terminal Cyan (#48BB78) used for success metrics and 'Live' status connectivity icons.
- Headline Font: 'Space Grotesk' (sans-serif) to convey a tech-forward, algorithmic aesthetic. Body Font: 'Inter' (sans-serif) for high legibility in complex data tables.
- Code Font: 'Source Code Pro' (monospace) for displaying sensitive MT5 account logins, server details, and passwords clearly.
- Thin-stroke technical icons (e.g., Candlestick charts, Shields, Pulsing nodes) reflecting a premium financial fintech feel.
- Subtle 30-second heartbeats for 'Live' data indicators and smooth progress bar transitions when metrics update from MetaApi.
